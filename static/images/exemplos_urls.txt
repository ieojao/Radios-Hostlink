DICAS IMPORTANTES:
- Sempre use URLs completas que começam com http:// ou https://
- Teste a URL no navegador antes de usar
- Imagens muito grandes podem demorar para carregar
- Alguns sites bloqueiam hotlinking (uso direto da URL)
- Para melhor performance, use imagens otimizadas (formato WebP ou JPEG comprimido)

PROBLEMAS COMUNS:
1. URL inválida ou quebrada
2. Site bloqueia hotlinking
3. Imagem muito pesada
4. Formato não suportado pelo navegador
5. Problemas de CORS (Cross-Origin Resource Sharing)
